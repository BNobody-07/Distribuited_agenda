# Distribuited_agenda

Agendamento Distribuido(Serviços/Consulta) em python

## Descrição

Este projeto implementa um sistema de agendamento distribuído em Python, composto por um servidor e um cliente. O cliente agora possui uma interface gráfica (GUI) desenvolvida com Tkinter, permitindo uma interação mais intuitiva e amigável.

## Funcionalidades do Cliente (GUI)

- **Listar Slots Disponíveis**: Exibe os horários disponíveis para agendamento.
- **Reservar um Slot**: Permite reservar um horário específico.
- **Listar Minhas Reservas**: Mostra as reservas feitas pelo usuário.
- **Cancelar uma Reserva**: Cancela uma reserva existente.
- **Visualizar Logs**: Exibe os logs de auditoria.
- **Apagar Dados**: Remove todos os dados de reservas e logs.

## Como Executar

1. Execute o servidor (`Servidor.py`).
2. Execute o cliente (`Cliente.py`) para abrir a interface gráfica.

## Requisitos

- Python 3.x
- Tkinter (incluído na instalação padrão do Python)
